# The Meadow — Симулятор сенокоса

## О проекте

Медитативный симулятор сенокоса в сеттинге якутского лета. Три столпа: **медитация**, **красота**, **уют**.

**Платформа:** Web (браузер)
**Движок:** Phaser 3 + Vite
**Режим:** Синглплеер (мультиплеер в планах)

## Структура проекта

```
src/
├── main.js              # Точка входа игры
├── scenes/
│   └── GameScene.js     # Основная игровая сцена
└── editor/
    ├── main.js          # Точка входа редактора
    ├── EditorScene.js   # Сцена редактора
    ├── MapGrid.js       # Рендер карты
    └── TilePalette.js   # Палитра тайлов

public/
├── assets/tilesets/     # Тайлсеты
└── maps/                # Карты (JSON)

docs/                    # Документация
```

## Ключевые механики

### Косьба (Timing System)
```
Шкала: [КРАСНЫЙ][ОРАНЖ][ЗЕЛЁНЫЙ][ОРАНЖ][КРАСНЫЙ]
Время:   0-200   200-400  400-600  600-800  800-1000 ms
```
Отпустить ЛКМ в зелёной зоне = Perfect.

### Время суток
- Полный цикл: 60 сек (тест) / 24 мин (прод)
- Уютные цвета: персиковый рассвет, янтарный закат, лавандовая ночь

### Слои карты
- **ground** — земля (фон, не интерактивный)
- **objects** — трава (можно косить)
