# The Meadow

Медитативный симулятор сенокоса. Якутское лето, бескрайние луга, уютная атмосфера.

## Запуск

```bash
npm install
npm run dev
```

Игра: http://localhost:3000/
Редактор карт: http://localhost:3000/editor.html

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD | Движение |
| ЛКМ (зажать) | Косить траву |

## Механика косьбы

Зажми ЛКМ и отпусти в нужный момент:

```
[КРАСНЫЙ][ОРАНЖ][ЗЕЛЁНЫЙ][ОРАНЖ][КРАСНЫЙ]
  Рано    Рано   PERFECT  Поздно  Поздно
```

Попади в зелёную зону для идеального скоса.

## Особенности

- Timing-based механика косьбы
- Плавная смена времени суток с уютными цветами
- Редактор карт с двумя слоями (земля + объекты)
- Сортировка по глубине — персонаж прячется в траве
